Contact List Application

HOW TO EXECUTE THE SYSTEM:
	1. Run source init in mysql console.
	2. Change directory to project and run python __init__.py
	3. Refer the Quick Start Guide to understand the how system works.

TECHNICAL REQUIREMENTS:
	1. OPERATING SYSTEM
		1. Ubuntu 18.04
			-i3 processor or higher
			-Minimum 500mb available RAM
		2. Windows 7/8/10
			-i3 processor or higher
			-Minimum 500mb available RAM

	2. SOFTWARE/PACKAGES PREREQUISITE:
		1. MySQL running on localhost
		2. Python 3.6 or above
		3. Python flask installed ( pip install Flask)
		4. Python MySQL (pip install MySQL-python)
		5. Python MySQL connector (pip install mysql-connector-python)
